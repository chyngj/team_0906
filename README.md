git사용 규칙
1. 브랜치 만들때 본인 성제외 영문 이름으로 
2. commit명 코멘트 어떤작업(CRUD) 어떤수정(어떤내용을 수정)을 했는지
3. 코드 파일 명 통일 자신이 작업할 함수 풀네임 소문자
4. pull했을때 다른사람의 코드에 이상이 있을 경우 말을 하고 고친 후 push한다
5. 자신이 작성한 코드를 git에 올릴 경우 다른사람의 변경상황을 pull한후 push한다.
